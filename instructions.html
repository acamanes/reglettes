Instructions générales

  • Une commande par ligne
  • Les majuscules / minuscules sont à respecter
  • Les lignes ne doivent pas commencer par des espaces
  • Le programme doit commencer par initialiser la liste des réglettes

Commandes autorisées

  • initialize(n) : Crée n réglettes triées par ordre décroissant de taille. La lettre n doit être remplacée par un nombre. Par exemple, initialize(5) permet de créer la liste suivante :
  • shuffle : Mélange les réglettes.
  • sort_even : Ordonne les réglettes dont la position est paire avec leur voisine de droite.
  • sort_odd : Ordonne les réglettes dont la position est impaire avec leur voisine de droite.
  • repeat n times {...} : Répète la série d'instructions entre les accolades n fois. La lettre n doit être remplacée par un nombre.
  • while exists_exchanges {...} : Répète la série d'actions entre accolades tant que des échanges ont été effectués dans cette série.

Exemple de code

initialize(10)
shuffle
repeat 3 times {
sort_even
sort_odd
}
shuffle
while exists_exchange {
sort_even
sort_odd
}
